== Description ==

Плагин Ultimate WordPress Cart Start является значительно облегченной версией плагина Ultimate WordPress Cart - профессионального инструмента для интернет-коммерции.
Несмотря на свою простоту, с его помощью Вы сможете продавать на своем блоге различные виды товаров, описания которых оформляются в виде обычных постов, а благодаря сервису EcommTools, в Вашем распоряжении будет многоуровневая партнерская программа и разнообразные маркетинговые средства (email-уведомления и рассылки, купоны скидок и дисконтные программы, upsell и т.д.).

= Основные возможности =
*	Продажа товаров любого типа;
*	Прием оплаты несколькими десятками различных способов;
*	Кнопки шорткодов для вставки кнопки добавления товара в корзину и вывода цены товара в описании;
*	Виджет оформления заказа (корзина);
*	Виджет проверки статуса заказа;
*	Виджет проверки купонов скидки;
*	Виджет входа в партнерские аккаунты (для участников Вашей партнерской программы);
*	Рекламный виджет для продвижения профессиональной версии этого плагина (Вы можете зарегистрироваться в его партнерской программе и зарабатывать, рекомендуя плагин своим посетителям);
*	Ссылка на видеоинструкцию по работе с плагином находится внутри (вкладка `О плагине`);
*	Плагин работает при наличии у Вас аккаунта любого вида (платного или бесплатного) в сервисе EcommTools – мощнейшем инструменте для управления Интернет-магазином и партнерской программой.

== Installation ==


1. Загрузине `Ultimate WordPress Cart Start` в директорию `/wp-content/plugins/` или воспользуйтесь стандартным загрузчиком WordPress, чтобы закачать архив на блог;
2. Активируйте плагин из меню `Плагины` в админпанели блога.
